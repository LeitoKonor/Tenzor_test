### About The Project

Test task for "Tensor" company

### Requerements:

- Python >= 3.10.6
- Selenium >= 4.11.2
- Pytest >= 7.4.0
- Poetry >= version 1.4.2

### Install:

git clone git@github.com:LeitoKonor/Tenzor_test.git

cd Tenzor_test

make start (to start the first test)

make start_2 (to start the second test)
